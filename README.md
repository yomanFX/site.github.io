# Show Dog Tycoon (Собачий Тайкун)

## Описание игры
Игра-симулятор и тайкон в 2D, где игроки разводят, воспитывают и соревнуются с породистыми собаками. Успех достигается через глубокое понимание генетики, стратегическое финансовое управление и мастерство на выставках собак. Конечная цель — построить всемирно известный питомник, разведя «идеальную» собаку и выиграв престижную мировую выставку собак.

## Основной игровой цикл
- Приобрести и воспитать: Игроки начинают с основных собак или покупают новых на динамическом рынке. Они должны управлять здоровьем и счастьем своих собак через ежедневный уход (еда, общение).
- Соревноваться (Выставка): Игроки регистрируют своих собак на выставках различных уровней. Успех — это сочетание врожденного счета Красоты (из генетики) и счета Навыков в мини-игре. Победа приносит призовые деньги и престиж.
- Размножение: Собаки с высокими оценками размножаются, чтобы передать свои превосходные генетические качества. Игроки должны стратегически подбирать пары собак для выращивания желательных черт (например, редкие цвета шерсти, высокая конформация), одновременно управляя риском наследственных дефектов.
- Управление (Тайкон): Игроки продают щенков на рынке по цене, определяемой генетическим качеством и редкостью. Они управляют своими финансами, реинвестируя прибыль в лучшую еду, выставочные сборы и приобретение новых кровных линий для улучшения своего питомника.

## Генетическая система
- Положительные атрибуты: У каждой собаки есть генетический балл (1-100) для ключевых признаков: Конформация, Качество шерсти, Темперамент, Выносливость.
- Генетические дефекты: Собаки могут нести скрытые рецессивные гены дефектов.
- Внутривидовая селекция: Разведение близко родственных собак резко увеличивает вероятность проявления дефектов у потомства.
